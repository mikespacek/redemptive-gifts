Google Sheets Cleanup Tool

Instructions:

  1. Open your Google Sheet
  2. Go to Extensions > Apps Script
  3. Replace the entire script with the content from google-apps-script-fixed.js
  4. Save the script
  5. Run the cleanupSpreadsheet function:
    • Select cleanupSpreadsheet from the function dropdown
    • Click the "Run" button (play icon)
  6. Deploy a new version as a web app:
    • Click on Deploy > New deployment
    • Select type: "Web app"
    • Set "Execute as" to "Me"
    • Set "Who has access" to "Anyone"
    • Click "Deploy"
    • Copy the new web app URL
  7. Update the NEXT_PUBLIC_GOOGLE_SHEET_URL in your .env.local file with the new URL
  8. Redeploy your application to Netlify

What This Will Fix:

The updated script will:

  • Properly handle form-urlencoded data that's causing the errors in your spreadsheet
  • Clean up existing error entries in your spreadsheet
  • Provide better debugging information for future submissions

Manual Cleanup

If you prefer to clean up the spreadsheet manually:

  1. Open your Google Sheet
  2. Look for rows where the timestamp column contains JSON data (starts with {"timestamp")
  3. Delete these rows

Testing After Cleanup

After cleaning up and updating the script:

  1. Complete a test submission at your website
  2. Check if the data appears correctly in your Google Sheet
  3. Verify that the scores match what's shown on the results page
